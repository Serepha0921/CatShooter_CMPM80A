## CMPM80A Final Project
### **Unity Version:** 2020.3.25f1 Intel 
### **Descritption:** A top-view shooter game which can play using only one hand.
### **Members:** 
    + Jae Choi 
    + Isha Chury 
    + Newton Novak 
    + Tara Roshan
---
### **How to Play:**
Keyboard:
- A: rotate left
- D: rotate right
- A+D: shoot
